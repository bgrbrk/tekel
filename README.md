# Turco Tekel POS (TR)

## Çalıştırma
```bash
python main.py
```
Giriş: **admin / admin** (kasiyer: **kasiyer / 1234**).  
Tam ekran: **F11**, ESC ile tam ekrandan çık.

- Arayüz Türkçe, tema **clam**, font **Segoe UI**.
- Büyük dokunmatik butonlar, otomatik ölçek.
- Kurlar paneli: API çalışmazsa **Kur yok**.
- Fiş yazdırma **varsayılan KAPALI** (Yönetici → Ayarlar).
- Stok & Sayım formlarında **Toptancı** seçimi var.
- Alış (QR) kaydında stok ve **cari** aynı anda güncellenir.
