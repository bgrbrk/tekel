# Turco Tekel POS — R.M Bilişim (TAM TÜRKÇE)
Giriş: kullanıcı **admin** — şifre **admin**

## Çalıştırma
```
cd TurcoTekelPos_TR_FINAL
python main.py
```

- Dokunmatik uyumlu, otomatik ölçek + tam ekran
- **Satış**: Barkod → Enter; +/− adet; **NAKİT / KREDİ KARTI**; Toplam TL ve altında **USD/EUR/GBP** eşdeğeri
- **Döviz Paneli**: Ana sayfanın sağ üstünde son kurlar
- **Alış (QR)**: Fatura QR/payload ile toplu ürün girişi, cari borca işler
- **Stok**: Liste + Barkod & tarih aralığı stok hareket raporu; **Ürün Ekle/Düzenle** penceresi
- **Toptancı Cari**: Türkçe rapor; **Ödeme Gir** ile cari ödeme, borç/alacak renklendirme
- **Kasa**: **Gelir/Gider** (not zorunlu) + tarih aralığı kasa raporu + Z raporu
- **Yönetici**: Fiş yazdır aç/kapat, varsayılan KDV, kullanıcı ekle, yedek alma
